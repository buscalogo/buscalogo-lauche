# Listing — English

## Name
BuscaLogo Agent

## Short description
Checks whether the current page is indexed by your local BuscaLogo Agent and lets you suggest adding it.

## Description

BuscaLogo Agent is the companion browser extension for the **BuscaLogo Agent** desktop app on your computer.

**Features**
- Shows if the current page is already indexed by your local Agent
- Suggest indexing with one click
- Talks only to `127.0.0.1:9970` — no cloud upload of browsing data

**Requirement**
You need BuscaLogo Agent running locally (panel at http://127.0.0.1:9970). Without it, the extension shows “Agent offline”.

**Privacy**
No browsing data is sent to remote BuscaLogo servers. See the privacy policy on our website.
