# Listagem — Português

## Nome
BuscaLogo Agent

## Resumo curto (132 caracteres máx. no Chrome)
Verifica se a página está indexada no BuscaLogo Agent local e sugere adicionar à busca colaborativa.

## Descrição

BuscaLogo Agent é a extensão companheira do **BuscaLogo Agent** instalado no seu computador.

**O que faz**
- Mostra se a página atual já está indexada no seu Agent local
- Permite sugerir a indexação com um clique
- Funciona offline em relação à nuvem: fala só com `127.0.0.1:9970`

**Requisito**
É necessário ter o BuscaLogo Agent rodando na máquina (painel em http://127.0.0.1:9970). Sem o Agent, a extensão indica “Agente offline”.

**Privacidade**
Nenhum dado de navegação é enviado a servidores remotos da BuscaLogo. Consulte a política de privacidade no site.

**Site**
https://buscalogo.com
